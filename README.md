# Hotel-Booking-Dashboard-Power-BI-Project-
This project is an interactive Hotel Booking Dashboard built using Power BI to analyze booking data, customer behavior, and revenue trends. It helps visualize key business insights for better decision-making in the hospitality industry.

Objectives:
Analyze hotel booking trends
Identify peak seasons and demand patterns
Track cancellations and booking status
Understand customer preferences
Monitor revenue performance

The dataset used in this project includes:
Booking ID
Customer details
Room type
Check-in & Check-out dates
Booking status (Confirmed / Cancelled)
Revenue and pricing

Tools & Technologies:
Power BI (.pbix)
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)
Data Visualization

Dashboard Features:
 Booking trends over time
 Cancellation analysis
 Room type performance
 Revenue insights
 Interactive filters and slicers

Key Insights:
Identified high-demand booking periods
Analyzed cancellation patterns
Found most popular room categories
Observed revenue fluctuations across months
